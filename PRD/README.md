# PRD – Vision produit

## Vision
Créer une plateforme interne ultra efficace pour une agence digitale,
centrée sur l’échange client, la validation et la production.

## Utilisateurs
- Agence (principal)
- Clients (accès limité)

## Piliers
- Inbox centrale (tout converge ici)
- Espaces clients modulaires
- Validation > gestion de tâches
- Documentation simple et structurée

## Ce que le produit N’EST PAS
- Un ClickUp bis
- Un Notion bis
- Un CRM
